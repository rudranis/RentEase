📞 QUICK HELP
═══════════════════════════════════════════════════════════════════════

Problem: "Localhost refused to connect"
Solution: Check if frontend npm run dev is running

Problem: "Cannot GET /api/listings"
Solution: Check if backend npm run dev is running

Problem: "MongoDB connection failed"
Solution: Check internet connection, IP whitelist in Atlas

Problem: "Email not received"
Solution: Configure Gmail credentials in backend/.env

Problem: "Images not uploading"
Solution: Configure Cloudinary credentials

For more help: See QUICKSTART.md Troubleshooting section
